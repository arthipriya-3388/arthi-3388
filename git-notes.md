# Git Fundamentals Notes

## Difference Between Git and GitHub

Git is a version control system used to track code changes.
GitHub is a platform used to store and share Git repositories online.

## What Does a Commit Do

A commit saves changes and creates a checkpoint in project history.

## What Is a Branch and Why Does It Exist

A branch allows developers to work on features separately without affecting the main code.

## What Is a Pull Request

A pull request is used to request merging changes from one branch into another.

## What Happens When You Push to GitHub

When we push, local commits are uploaded from our computer to GitHub.
